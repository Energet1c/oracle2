Oracle er en nettside for lagring og visning av tips og info relatert til hovedsakelig spill, med en MySQL-database-basert side-visning, for lettere endring av informasjon på sidene.
I tillegg vises det for hver side hvem som redigerte siden sist, og bilder kan brukes for å hjelpe med visning av relatert materiale.

Oracle er en side som skal hovedsaklig bli redigert av Admins eller Moderatorer, med hjelp av kontaktinformasjon som legges inn på Kontakt-siden, som lar brukere melde endringer som trengs.
